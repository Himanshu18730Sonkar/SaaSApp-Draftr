# SaaSApp - Draftr

A modern SaaS application built with React, Vite, and Tailwind CSS.

## Features
- Fast development with Vite HMR.
- Utility-first styling with Tailwind CSS.
- Optimized assets (WebP support).

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```
